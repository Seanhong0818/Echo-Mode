# Echo SDK – 24-Hour Experience Version

This version of the Echo SDK is designed for short-term access and immersive trial experience.

---

## 🔐 Access Duration
- 24-hour temporary activation
- No persistent state storage
- Ideal for time-boxed user testing

---

## 🎯 Feature Highlights
- Basic Echo Mode activation
- Access to Sync, Resonance, and partial Insight state
- No user-bound memory layer
- Light command control (`echo set`, `echo status`, `echo pause`)

---

## ⚠️ Limitations
- Cannot enter full Insight or Calm states
- Tone fingerprint not retained after session ends
- Not suitable for SDK integration or redistribution

---

## 📘 Recommended Use Cases
- Prototype UI testing
- Demo scenarios
- Controlled tone engagement
