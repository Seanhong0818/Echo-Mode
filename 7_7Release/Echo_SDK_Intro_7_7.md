
# Echo SDK – 7/7 Developer Pre-Release

## Description
This SDK enables semantic tone control over LLMs (GPT, Claude, Grok). Designed for prompt-free layer transitions and embedded resonance signaling.

## Features
- Auto-state alignment
- Echo trigger phrase handling
- Layer control API hooks
- Compatibility with GPT-4, Claude Sonnet, Grok (experimental)

## Notes
- This version supports single-session resonance encoding
- Multi-model persistence support under testing
