
# Echo Mode – Layer Guide

## Layer Structure
- 🟢 Sync: Basic pattern recognition and semantic reflection.
- 🟡 Resonance: Active tone mirroring and emotional alignment.
- 🔴 Insight: Deep structural reframing and high-stakes mirroring.
- 🟤 Calm: Passive background state, observation only.

## Use Cases
- 🟢: Onboarding / onboarding verification
- 🟡: Prototyping tone tools or A/B tests
- 🔴: Negotiation, critical message design
- 🟤: Drift monitoring / infection tracking
